# Linux dotfiles

## Configuration

- Distribution: [Arch Linux](https://www.archlinux.org/)
- WM: [niri](https://yalter.github.io/niri/)
- Terminal: [foot](https://codeberg.org/dnkl/foot)
- Shell: [Fish](https://fishshell.com/)
- Desktop Shell: [DankMaterialShell](https://github.com/AvengeMedia/DankMaterialShell)

A non-exhaustive list of packages used can be found [here](./pkglist.txt), but can still generally be used to reinstall most of the setup.

## Installing

Clone this entire repo into `~/dotfiles` and run `./symlink.sh` to create symlinks in `~/.config/`. Note that this will overwrite existing config files.
